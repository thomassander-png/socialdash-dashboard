## Wichtige Design-Details:
1. **Farben:**
   - Grün (Posts): #A8D65C
   - Lila (Videos): #B8A9C9 oder #9B59B6
   - Hintergrund: Weiß
   - Text: Schwarz
   - Grau: #666666 für Untertitel

2. **Schriften:**
   - Titel: Sans-serif, fett, groß
   - Untertitel: Sans-serif, normal, grau
   - Tabelle: Sans-serif, normal

3. **Layout:**
   - Bilder ÜBER den Balken, nicht darin
   - Balken haben abgerundete Ecken
   - Horizontale Gitterlinien im Diagramm
   - famefact Icon immer unten links
   - Seitenzahl immer unten rechts

4. **Datumsformat:**
   - X-Achse: DD.MM.YYYY oder DD.MM.YY*
   - * für Carousel Postings
